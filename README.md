# LaunChar
タイピングするたび画面に文字が飛び出すデスクトップアクセサリ  

## 動作環境
動作には .NET Framework 4.5 以上が必要です  
その上で Windows 7/10 での動作確認をしました  
８でも動くと思います  

## 使い方
![LaunChar.gif](https://github.com/seinosuke/LaunChar/blob/master/LaunChar.gif)
起動するだけです  
終了する場合はタスクバーからメインウィンドウを閉じてください
